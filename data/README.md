# `data/`

Star-schema tables as **local Parquet**, produced by the notebooks in `../notebooks/`.
Read with `pd.read_parquet`, written with `df.to_parquet(path, index=False)`.
Migration path to Microsoft Fabric: swap to `abfss://` + `spark.read.parquet` — schema is unchanged.

## Dimensions

| File | Key | Produced by | Notes |
|---|---|---|---|
| `dim_nfl_players.parquet` | `gsis_id` | 05 | Full nflverse registry; primary FK for all facts |
| `dim_rookie_prospect.parquet` | `player_key` | 01, 08x, 08z | Current draft class; pre-signing proxy for `gsis_id` |
| `dim_fantasy_teams.parquet` | `team_key` | 03 | 28 league teams + conference/cap metadata |
| `dim_contract.parquet` | `contract_id` | 02 | Contract types driving cap-hit % and dead money |
| `dim_nfl_teams.parquet` | `team_abbr` | 04 | NFL team metadata |
| `dim_position.parquet` | `position_raw` | 01 | Raw → canonical position transformer (+ `side_of_ball`) |
| `dim_school.parquet` | `school_raw` | 01 | Raw → canonical school + conference transformer |
| `dim_player_alias.parquet` | `name_clean + position_raw` | 08y, 08z | Persistent fuzzy-match decisions (variant name → `player_key`) |
| `dim_fantrax_crosswalk.parquet` | `scorer_id` | 09a | Fantrax `scorer_id` → `gsis_id` + `player_key` |

## Facts

| File | Grain / key | Produced by | Notes |
|---|---|---|---|
| `fact_rookie_rankings.parquet` | `player_key + source_name + phase + draft_year` | 08, 08a–08e | Expert rankings, 10 sources, phase cascade |
| `fact_fantrax_adp.parquet` | `scorer_id + season + week` | 09 | Weekly Fantrax ADP/salary; offense (ADP) + IDP (roster) |
